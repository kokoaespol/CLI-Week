CLI Week 2023

Welcome to the CLI Week website!

The CLI Week is an event in which participants have one week to have fun and share which each other while they build an awesome CLI tool.

During this week, participants will also be able to assist to the different talks and workshops that will be hosted. More info on that here.

At the end of the week a jury will select the "best" 3 projects and the winners will receive a KokoCertificate.

Remember! The most important thing is to have fun creating your project.

How to participate?

Wanna write code? Visit the CLI Week repository on Github for instructions on how to get started! You can also register in this form if you want.

Register here.
